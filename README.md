# VitaCare

VitaCare é uma plataforma desenvolvida para conectar pacientes e nutricionistas por meio de um acompanhamento nutricional mais inteligente, organizado e personalizado.

## Visão geral do projeto

O VitaCare foi pensado para:
- pessoas que desejam melhorar a alimentação e buscam apoio profissional
- nutricionistas que querem ampliar seus atendimentos com um acompanhamento mais personalizado

A proposta da plataforma é oferecer uma pré-análise inteligente com base no histórico, progresso e informações do usuário, tornando o cuidado nutricional mais eficiente e direcionado.

## O que este projeto inclui

- Landing page responsiva desenvolvida com React e TypeScript
- Estilização com Tailwind CSS
- Animações com Framer Motion
- Estrutura de componentes organizada
- Separação por seções para facilitar o trabalho em equipe

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Router
- TanStack Query

## Como executar localmente

```bash
npm install
npm run dev
```

Depois, abra no navegador o endereço exibido no terminal.

## Estratégia de branches

- `main`: estrutura base do projeto
- `as-landing-page`: desenvolvimento da Hero Section, seção de Funcionalidades e seção Como Funciona
- `mm-landing-page`: desenvolvimento das seções Nutricionistas, Depoimentos, Call to Action e Footer
